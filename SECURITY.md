# Security Policy

## Supported projects

Security fixes and responsible disclosure apply to repositories actively maintained by the Unimals organization.

As a default guideline:

- **Actively maintained** repositories may receive security fixes and coordinated disclosure support
- **Archived or experimental** repositories may not receive timely patches

Please check each repository's README or activity status for project-specific support details.

## Reporting a vulnerability

If you believe you have found a security vulnerability in a Unimals project, please report it responsibly.

### Please include

- affected repository / package / service
- vulnerability type
- clear reproduction steps
- impact assessment
- proof of concept if available
- any suggested mitigation

### Please do not

- open a public GitHub issue for unpatched vulnerabilities
- disclose the issue publicly before we have had a reasonable chance to investigate and respond
- access, modify, or exfiltrate data beyond what is necessary to demonstrate the issue

## How to report

For now, please report vulnerabilities privately by contacting the organization maintainers directly.

Recommended temporary process:

- open a **private security advisory** on the affected repository, if enabled
- or contact the organization owner through a private channel

If a dedicated security email is added later, this policy should be updated to point to that address.

## Response expectations

We aim to:

- acknowledge receipt within a reasonable time
- validate and triage the report
- work on mitigation or remediation based on severity and impact
- coordinate disclosure when appropriate

Response time is best-effort and may vary by repository status, severity, and maintainer availability.

## Safe harbor

We support good-faith security research intended to improve the safety of our systems and users.

If you act in good faith, avoid privacy violations or service disruption, and give us a reasonable opportunity to respond before public disclosure, we will treat your report as responsible disclosure.
