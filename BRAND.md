# Unimals Brand Direction

## Core idea
Unimal = Unique + Animal.

Unimals is a builder-first brand for practical AI agents, automation systems, and execution tooling.

## Brand traits
- sharp
- calm
- operational
- inventive
- slightly wild, never sloppy

## Visual direction
- Dark base palette with high-contrast accents
- Clean, minimal layout
- Animal energy without looking childish
- Futuristic, but grounded and usable

## Logo direction
- Wordmark: Unimals
- Optional icon: abstract claw / paw / horn hybrid
- Avoid mascots that feel cartoonish
- Should work in monochrome first

## Banner direction
- Wide dark background
- Minimal composition
- Subtle texture, grid, or motion lines
- One strong line of copy

## Suggested copy options
1. Practical AI agents for real-world execution.
2. Build agents. Ship systems. Compound leverage.
3. From ideas to execution.

## Profile consistency
- Org description should stay concise and utility-first
- README can carry the fuller narrative
- Logo and banner should communicate seriousness with character
